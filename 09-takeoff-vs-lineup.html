Take-Off vs Line-Up

An aircraft is given Take-Off while another aircraft is given Line-Up on the same runway

TAKE-OFF vs LINE-UP

📊 Data Required

Clearances Assigned Runway Surveillance Holding Point

⚠️ General Rule

Important: In each scenario, it is deemed that the first Clearance in the heading title is the one that has been input by the Controller first and the second Clearance triggers the alert.

🚨 Alert Triggered

1 Line-Up from Same Direction Holding Point

If BAW456 is given Take-Off and IBE987 (or AFR123) is given Line-Up from a Holding Point on the same runway.

⚠ CONFLICT

Situation 1: BAW456 given Take-Off while IBE987 (or AFR123) is given Line-Up from a Holding Point on the same runway

2 Line-Up from Opposite Direction Holding Point

If BAW456 is given Take-Off and KLM789 is given Line-Up from a Holding Point on the same runway in the opposite direction.

⚠ CONFLICT

Situation 2: BAW456 given Take-Off while KLM789 is given Line-Up from the opposite direction on the same runway

✅ Exemptions to the Rule

  • 1
    Holding point is needed to determine whether the position of IBE987 (or AFR123) is behind the position of BAW456 (based on surveillance), in which case ✓ NO ALERT is triggered.
  • 2
    Surveillance is needed to determine whether BAW456 is airborne (positive climb), in which case ✓ NO ALERT is triggered for KLM789.

📋 Quick Reference

Situation Description Alert Exemption
1 Line-Up from same direction Holding Point CONFLICT Line-Up position is behind Take-Off position
2 Line-Up from opposite direction Holding Point CONFLICT Take-Off aircraft is airborne (positive climb)
